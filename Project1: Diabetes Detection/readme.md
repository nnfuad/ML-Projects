## üéØ Overview

This project implements a Feedforward Neural Network (FNN) to predict the likelihood of diabetes based on diagnostic measurements. The model is trained on the Pima Indians Diabetes Database and demonstrates the application of deep learning in healthcare diagnostics.

**Key Features:**
- Data preprocessing and normalization
- Feedforward Neural Network implementation
- Model evaluation and visualization
- Hyperparameter tuning capabilities


## üìä Dataset

The project uses the open-source dataset from Kaggle.

**Features:**
- Pregnancies: Number of times pregnant
- Glucose: Plasma glucose concentration
- BloodPressure: Diastolic blood pressure (mm Hg)
- SkinThickness: Triceps skin fold thickness (mm)
- Insulin: 2-Hour serum insulin (mu U/ml)
- BMI: Body mass index
- DiabetesPedigreeFunction: Diabetes pedigree function
- Age: Age in years

**Target:**
- Outcome: 0 (No diabetes) or 1 (Diabetes)
